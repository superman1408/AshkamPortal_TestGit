# ERP Management System

A complete ERP (Enterprise Resource Planning) web application developed for managing employee operations and organizational workflows within the company.

## Overview

This ERP system helps organizations manage daily employee activities, attendance tracking, leave management, communication, payroll handling, and departmental operations from a centralized platform.

The application is designed to improve productivity, simplify HR processes, and provide efficient management of employee records and internal communication.
